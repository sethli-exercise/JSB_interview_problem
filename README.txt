This project can be run by running:
- ollama locally ("ollama serve" in the terminal)
- the streamlit application locally ("streamlit run <path to streamlit_app.py>" in the terminal)

The project will take any prompt and produce some response.
It can speak the last response back to the user.
It can also take your voice instead of text as input.
